# AI 写作

## 概念说明

AI 写作工具利用大语言模型帮助用户生成、改写、润色各类文本内容。从营销文案到商务邮件，从技术文档到学术论文润色，AI 写作已成为知识工作者的核心提效手段。关键在于掌握正确的使用方法，让 AI 成为写作的"副驾驶"而非替代者。

### AI 写作能力图谱

| 写作类型 | AI 擅长度 | 说明 |
|----------|-----------|------|
| **营销文案** | ⭐⭐⭐⭐⭐ | 多风格切换、批量生成、A/B 测试 |
| **商务邮件** | ⭐⭐⭐⭐⭐ | 格式规范、语气得体、多语言 |
| **技术文档** | ⭐⭐⭐⭐ | 结构清晰，但需人工校验技术细节 |
| **报告总结** | ⭐⭐⭐⭐⭐ | 信息提取和结构化能力强 |
| **翻译润色** | ⭐⭐⭐⭐ | 流畅度高，专业术语需校对 |
| **创意写作** | ⭐⭐⭐ | 可提供灵感，但缺乏真正的创造力 |
| **学术写作** | ⭐⭐⭐ | 可辅助结构和润色，核心观点需原创 |

## 文案生成

### 营销文案

AI 在营销文案领域表现出色，能快速生成多种风格的文案。

**Prompt 模板 — 产品描述：**
```
你是一位资深的营销文案专家。请为以下产品撰写营销文案：

产品：[产品名称]
目标受众：[目标用户画像]
核心卖点：[1-3 个核心卖点]
文案风格：[专业/轻松/幽默/高端]
文案长度：[短文案 50 字 / 中文案 150 字 / 长文案 300 字]
使用场景：[朋友圈/小红书/淘宝详情页/公众号]

请生成 3 个不同角度的文案版本。
```

**Prompt 模板 — 社交媒体内容：**
```
请为 [品牌/产品] 生成一周的社交媒体内容计划：

平台：小红书
主题方向：[主题]
内容风格：[种草/教程/测评/日常分享]
每篇要求：标题（含 emoji）+ 正文（200-300 字）+ 标签（5-8 个）

请确保内容符合小红书的社区调性。
```

### 多风格切换技巧

| 风格 | Prompt 关键词 | 适用场景 |
|------|--------------|----------|
| 专业严谨 | "专业、数据支撑、权威" | B2B 营销、白皮书 |
| 轻松活泼 | "口语化、有趣、接地气" | 社交媒体、种草文 |
| 高端大气 | "简洁、高级感、留白" | 奢侈品、高端品牌 |
| 情感共鸣 | "走心、故事化、引发共鸣" | 品牌故事、公益传播 |
| 紧迫感 | "限时、稀缺、行动号召" | 促销活动、限时优惠 |

## 邮件撰写

### 商务邮件

**Prompt 模板 — 商务邮件：**
```
请帮我撰写一封商务邮件：

场景：[初次联系/跟进/感谢/道歉/邀请/拒绝]
收件人：[职位/关系]
核心内容：[邮件要传达的关键信息]
语气：[正式/半正式/友好]
语言：[中文/英文/中英双语]

要求：
- 主题行简洁明了
- 正文结构清晰（开头问候 → 正文 → 行动号召 → 结尾）
- 语气得体专业
```

**常用邮件场景 Prompt：**

```
# 会议邀请
请撰写一封会议邀请邮件，邀请团队成员参加下周三下午 2 点的项目评审会。
包含：会议目的、议程、需要准备的材料、会议链接。

# 项目进度汇报
请撰写一封项目进度汇报邮件给上级领导。
项目名称：[名称]
本周完成：[完成事项]
下周计划：[计划事项]
风险/问题：[如有]
需要支持：[如有]

# 礼貌拒绝
请帮我撰写一封礼貌拒绝合作邀请的邮件。
拒绝原因：[原因]
要求：语气友好但明确，保留未来合作可能性。
```

## 报告总结

### 会议纪要

**Prompt 模板：**
```
请根据以下会议记录整理会议纪要：

[粘贴会议记录/转录文本]

输出格式：
1. 会议基本信息（时间、参会人、主题）
2. 讨论要点（按议题分类）
3. 决议事项（明确责任人和截止日期）
4. 待办事项清单（TODO List）
5. 下次会议安排
```

### 周报/月报

**Prompt 模板：**
```
请根据以下工作内容整理周报：

本周工作内容：
[列出本周完成的工作]

输出格式：
1. 本周工作总结（按项目/模块分类）
2. 关键成果和数据
3. 遇到的问题和解决方案
4. 下周工作计划
5. 需要协调的事项

要求：语言简洁专业，突出成果和价值。
```

### 文档摘要

**Prompt 模板：**
```
请对以下文档进行摘要：

[粘贴文档内容]

要求：
1. 一句话总结（30 字以内）
2. 核心要点（3-5 条）
3. 关键数据和结论
4. 行动建议（如适用）
```

## 翻译润色

### AI 翻译对比

| 工具 | 优势 | 劣势 | 适用场景 |
|------|------|------|----------|
| **ChatGPT/Claude** | 上下文理解强、可指定风格 | 长文本可能不一致 | 灵活翻译、风格化翻译 |
| **DeepL** | 翻译质量高、术语一致 | 不支持中文（部分） | 欧洲语言翻译 |
| **Google 翻译** | 语种多、速度快 | 质量一般 | 快速理解大意 |
| **有道翻译** | 中文优化好 | 专业领域一般 | 日常中英翻译 |
| **沉浸式翻译** | 双语对照、网页翻译 | 依赖底层模型 | 网页/文档阅读 |

### 翻译 Prompt 技巧

**学术论文翻译：**
```
请将以下英文论文摘要翻译为中文：

[粘贴英文摘要]

要求：
1. 保持学术论文的正式语体
2. 专业术语保留英文原文并在括号中标注
3. 确保逻辑关系准确传达
4. 长句适当拆分，符合中文表达习惯
```

**技术文档翻译：**
```
请将以下技术文档翻译为中文：

[粘贴技术文档]

要求：
1. 技术术语使用业界通用的中文翻译
2. 代码和命令保持原文不翻译
3. 保持原文的格式结构
4. 对于没有通用中文翻译的术语，保留英文
```

**润色 Prompt：**
```
请润色以下中文文本：

[粘贴原文]

润色要求：
1. 修正语法和标点错误
2. 优化句式结构，使表达更流畅
3. 保持原文的核心意思不变
4. 风格：[学术/商务/轻松]
5. 标注修改的地方并说明修改理由
```

## 实战要点

### AI 写作最佳实践

1. **明确目标**：写作前明确目的、受众、风格、长度
2. **提供上下文**：给 AI 足够的背景信息和参考材料
3. **迭代优化**：初稿 → 反馈 → 修改 → 润色，多轮迭代
4. **人工审核**：AI 生成内容必须经过人工审核和修改
5. **保持原创**：用 AI 辅助而非替代，保持个人风格和观点

### 常见陷阱

- **AI 味过重**：生成内容过于模板化，缺乏个性
- **事实错误**：AI 可能编造数据或引用，需要核实
- **风格不一致**：长文中前后风格可能不统一
- **过度依赖**：完全依赖 AI 会导致写作能力退化
