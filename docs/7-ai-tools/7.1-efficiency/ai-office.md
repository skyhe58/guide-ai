# AI 办公

## 概念说明

AI 办公工具将大语言模型能力嵌入日常办公场景，覆盖 PPT 制作、Excel 数据处理、文档总结、会议纪要等高频任务。这些工具的核心价值在于将重复性的格式化工作交给 AI，让人专注于内容和决策。

### AI 办公能力矩阵

| 办公场景 | AI 能力 | 提效倍数 |
|----------|---------|----------|
| **PPT 制作** | 从大纲自动生成演示文稿 | 5-10x |
| **Excel 处理** | 自然语言生成公式和图表 | 3-5x |
| **文档总结** | 长文档自动摘要和要点提取 | 5-10x |
| **会议纪要** | 语音转文字 + 自动整理 | 10x+ |
| **邮件管理** | 智能分类、自动回复草稿 | 2-3x |

### PPT 生成实操指南

**方法一：AI 对话 → PPT 大纲 → AI PPT 工具**

```
# 第一步：用 ChatGPT/Claude 生成大纲
请为以下主题生成 PPT 大纲：

主题：[演讲主题]
受众：[目标受众]
时长：[演讲时长]
风格：[商务/学术/创意]

要求：
1. 包含封面页、目录页、内容页、总结页
2. 每页标题 + 3-5 个要点
3. 标注需要配图或图表的页面
4. 总页数控制在 [数量] 页以内

# 第二步：将大纲导入 Gamma/美图 AI PPT 生成
```

**方法二：直接使用 AI PPT 工具**

```
# Gamma AI 使用流程
1. 访问 gamma.app
2. 选择 "Generate" → 输入主题
3. 选择演示文稿类型和风格
4. AI 自动生成完整 PPT
5. 逐页调整内容和设计
6. 导出为 PPTX/PDF
```

## Excel 公式与数据分析

### AI 生成 Excel 公式

**使用 ChatGPT/Claude 生成复杂公式：**

```
# 场景：数据分析
请帮我写一个 Excel 公式：
- 数据在 A 列（日期）和 B 列（销售额）
- 需要计算最近 30 天的移动平均值
- 结果放在 C 列

# AI 回答示例
=AVERAGEIFS(B:B, A:A, ">="&(TODAY()-30), A:A, "<="&TODAY())
```

**常用 Excel AI 场景：**

| 场景 | Prompt 示例 |
|------|------------|
| 复杂公式 | "写一个 VLOOKUP 嵌套 IF 的公式，根据条件匹配数据" |
| 数据透视 | "如何用数据透视表分析每月各产品的销售趋势" |
| 图表建议 | "这组数据适合用什么图表展示？请给出建议" |
| VBA 脚本 | "写一个 VBA 宏，自动合并多个工作表的数据" |
| 数据清洗 | "如何批量去除单元格中的空格和特殊字符" |

## 实战要点

### AI 办公工具组合方案

**日常办公组合（免费方案）：**
- PPT：讯飞智文 / 美图 AI PPT
- Excel：ChatGPT 生成公式
- 文档：Kimi / 通义千问
- 会议：通义听悟
- 写作：DeepSeek

**专业办公组合（付费方案）：**
- PPT：Gamma AI + ChatGPT 大纲
- Excel：Microsoft Copilot
- 文档：Claude Projects
- 会议：飞书妙记
- 写作：ChatGPT Plus

### 效率提升技巧

1. **建立 Prompt 库**：将常用的办公 Prompt 保存为模板，随时调用
2. **批量处理**：利用 AI 批量生成周报、邮件、文档摘要
3. **工作流自动化**：会议录音 → 自动转录 → AI 整理 → 发送纪要
4. **模板标准化**：用 AI 生成标准化的文档模板，统一团队输出格式

## 注意事项

- **数据安全**：公司机密文档不要上传到公共 AI 工具
- **格式兼容**：AI 生成的 PPT 导出后可能需要微调格式
- **内容审核**：AI 生成的数据和图表需要人工核实准确性
- **版权合规**：注意 AI 生成内容在商业场景中的版权问题
